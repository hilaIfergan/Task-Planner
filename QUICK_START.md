# התחלה מהירה - Task Planner PWA

## 🚀 התקנה מהירה (5 דקות)

### שלב 1: העלה ל-GitHub Pages (הכי קל!)

1. **צור repository חדש ב-GitHub**
   - לך ל [github.com](https://github.com)
   - לחץ על "New repository"
   - שם: `task-planner` (או כל שם שתרצה)
   - לחץ "Create repository"

2. **העלה את הקבצים**
   - פתח את תיקיית `PWA` במחשב
   - גרור את כל הקבצים ל-GitHub (או השתמש ב-Git)
   - לחץ "Commit changes"

3. **הפעל GitHub Pages**
   - לך ל-Settings → Pages
   - Source: בחר `main` branch
   - Folder: `/ (root)`
   - לחץ Save
   - חכה 2 דקות...

4. **האפליקציה זמינה!**
   - הכתובת תהיה: `https://yourusername.github.io/task-planner`

### שלב 2: הוסף לאייפון

1. **פתח Safari באייפון**
2. **נווט לכתובת** מה-GitHub Pages
3. **לחץ על כפתור השיתוף** (⬆️)
4. **בחר "הוסף למסך הבית"**
5. **לחץ "הוסף"**
6. **✅ מוכן!** האפליקציה על המסך!

## 📱 שימוש

- **לחץ על +** כדי להוסיף משימה
- **לחץ על המשימה** כדי לערוך
- **לחץ על העיגול** כדי לסמן הושלמה
- **לחץ על "תכנן מחדש"** כדי לדחות

## ⚠️ הערות חשובות

1. **אייקונים** - צריך ליצור `icon-192.png` ו-`icon-512.png` (אופציונלי)
2. **HTTPS** - GitHub Pages מספק HTTPS אוטומטית ✅
3. **תזכורות** - צריך לאשר הרשאות כשהאפליקציה תבקש

## 🎨 יצירת אייקונים (אופציונלי)

אם תרצה אייקונים מותאמים אישית:

1. צור תמונה 512x512 פיקסלים
2. שמור כ-`icon-512.png`
3. צמצם ל-192x192 ושמור כ-`icon-192.png`
4. העלה ל-GitHub

**או** השאר בלי אייקונים - האפליקציה תעבוד גם ככה!

## ❓ בעיות?

- **לא נטען?** - ודא שכל הקבצים ב-root של ה-repository
- **תזכורות לא עובדות?** - Settings → Safari → Notifications
- **נתונים נעלמו?** - בדוק ב-Settings → Safari → Clear History
