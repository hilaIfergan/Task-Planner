# מדריך התקנה מפורט - Task Planner PWA

## שלבים להתקנה באייפון

### שלב 1: העלאת הקבצים לשרת

יש לך כמה אפשרויות:

#### אפשרות A: GitHub Pages (חינם)
1. צור repository חדש ב-GitHub
2. העלה את כל הקבצים מתיקיית `PWA` ל-repository
3. לך ל-Settings → Pages
4. בחר branch `main` ו-save
5. האפליקציה תהיה זמינה בכתובת: `https://yourusername.github.io/repository-name`

#### אפשרות B: Netlify (חינם)
1. היכנס ל-[netlify.com](https://netlify.com)
2. גרור את תיקיית `PWA` לאתר
3. האפליקציה תהיה זמינה מיד בכתובת: `https://random-name.netlify.app`

#### אפשרות C: שרת משלך
1. העלה את הקבצים לשרת Web שלך
2. ודא שהאתר על HTTPS (חובה ל-PWA)

### שלב 2: הוספה למסך הבית באייפון

1. **פתח Safari באייפון** (לא Chrome או דפדפנים אחרים)
2. **נווט לכתובת** של האפליקציה (מה-GitHub Pages/Netlify/שרת)
3. **לחץ על כפתור השיתוף** (החץ למעלה בתחתית המסך)
4. **גלול למטה** ובחר **"הוסף למסך הבית"**
5. **שנה את השם** אם תרצה (אופציונלי)
6. **לחץ "הוסף"**
7. **האפליקציה תופיע על המסך** כמו אפליקציה רגילה!

### שלב 3: הרשאות

כשתפתח את האפליקציה בפעם הראשונה:
1. **תזכורות** - האפליקציה תבקש הרשאה לשלוח תזכורות
2. **לחץ "אשר"** כדי לקבל תזכורות למשימות

## שימוש באפליקציה

### יצירת משימה חדשה
1. לחץ על כפתור ה-+ (בפינה הימנית העליונה)
2. מלא את פרטי המשימה
3. בחר תאריך ושעה (אופציונלי)
4. הגדר תזכורת (אופציונלי)
5. לחץ "שמור"

### סימון משימה כהושלמה
- לחץ על העיגול ליד המשימה

### עריכת משימה
- לחץ על המשימה עצמה

### מחיקת משימה
- לחץ על "מחק" בתחתית המשימה

### תכנון מחדש
- לחץ על "תכנן מחדש" בתחתית המשימה
- הכנס תאריך ושעה חדשים

## פתרון בעיות

### האפליקציה לא נטענת
- ודא שיש חיבור לאינטרנט (לפחות בפעם הראשונה)
- נסה לרענן את הדף
- בדוק שהכתובת נכונה

### תזכורות לא עובדות
1. לך ל-**Settings** → **Safari** → **Notifications**
2. ודא ש-**Task Planner** מופיע ברשימה
3. הפעל את ההתראות

### הנתונים לא נשמרים
- האפליקציה שומרת הכל מקומית במכשיר
- הנתונים לא ייעלמו גם אם תמחק את הדפדפן
- כדי למחוק הכל: Settings → Safari → Clear History and Website Data

### האפליקציה לא מופיעה כמו אפליקציה
- ודא שהוספת דרך Safari (לא דפדפנים אחרים)
- ודא שהאתר על HTTPS
- נסה למחוק ולהוסיף מחדש

## הערות חשובות

✅ **כל הנתונים נשמרים מקומית** - אין צורך בחיבור לאינטרנט  
✅ **עובדת offline** - אחרי הטעינה הראשונה  
✅ **לא דורשת App Store** - הוספה ישירה למסך הבית  
✅ **לא דורשת Mac** - הכל דרך הדפדפן  

## צור אייקונים

לפני העלאת האפליקציה, צריך ליצור אייקונים:
- `icon-192.png` - 192x192 פיקסלים
- `icon-512.png` - 512x512 פיקסלים

אפשר להשתמש ב:
- [Canva](https://canva.com) - עיצוב אייקון
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator) - יצירה אוטומטית

או פשוט להשאיר את הקבצים ריקים - האפליקציה תעבוד גם בלי אייקונים.

## שאלות?

אם יש בעיות או שאלות, בדוק את:
- Console של הדפדפן (F12 או Cmd+Option+I)
- Application → Service Workers (ב-DevTools)
- Application → IndexedDB (לראות את הנתונים)
